# TimerPro SaaS 商家使用白皮书与保姆级操作手册

欢迎使用 TimerPro SaaS 智能计时收银系统。本手册将带您从零开始，快速配置符合您店铺的专属计费模型，并掌握日常店务操作。

---

## 第一章：基础开通与设置

### 1.1 注册并登录
1. 访问系统提供的 `register.html`。
2. 填写您的**店铺名称**、**负责人姓名**以及您的**邮箱**。
3. 点击获取验证码，查收邮件并完成验证。
4. 注册成功后，前往 `login.html` 登录系统。

### 1.2 计费核心参数解释（进入系统后右上角点击⚙️系统设置）
这是系统的“大脑”。即使是最复杂的实体店计费规则，在这里也能实现。

**基础参数**：
- **通用基准价 (元/小时)**：您店里的默认收费标准（例如：20元/小时）。如果不使用复杂算法，就会一直按这个单价和客户时间折算。
- **基准时长 (分钟)**：默认请填 `60`。
- **加时单价 (元/小时)**：当客人买了2小时的固定套餐，结果玩了3小时，这多出来的1小时怎么收费？填在这里。（也可填同上）。
- **计费缓冲 (分钟)**：容错时间。例如设置为 `10`，客人在规定时间外多玩了9分钟，系统不会多收一分钱。这能大大减少前台与客人的纠纷。

## 第二章：进阶！解密“阶梯计速参数”

如果您想实现类似于：“每30分钟收15块钱，不到30分钟也按30分钟算”的硬核逻辑，请将**算法模式**切换为【阶梯计价(按块)】。

参数白话文对照表：
* **计费块大小 N (分)**：计费的“一个周期”。例如填 `30`，代表每半小时扣一次费。
* **每块收费 Y (元)**：在这个周期内我们要收多少钱。例如填 `15`。
* **进位分割系数 K**：高级自定义。默认填 `1`，无需更改。
* **抹零免单界限 X (分)**：每个新周期开始的容错界限。例如：X定为 `5`。当客人玩了 32 分钟结账。这多出来的2分钟落在5分钟界限内，此时直接免单这一个新块，只收前30分钟的15元。

## 第三章：上架团购套餐

在【系统设置】最下方，您可以无限添加您在大众点评、美团售卖的团购方案。
系统支持的三种终极套餐形态：

1. **计时长（如定额限时单人套餐）**：只需填写包含时长，例如内含 `120`（代表2小时）。超过2小时会自动触发您配置的“加时单价”。
2. **包时间段（如：早鸟特惠 10:00-14:00）**：需设定“时段起”和“时段止”。客人只要用这个套餐，不管几点来，14:00之后就会开始加收超时费。
3. **全天畅玩**：不需要填时长。买断制，从客人进门一直玩到打烊，一口价。

## 第四章：前台每日工作流实操

### 4.1 迎接客人 (开单)
在【收银台】左下角：
1. **标识号**：填入手牌号、手机尾号或客人特征均可。
2. **开单模式**：
   - 如果客人没买套餐，选**先玩后付**。
   - 如果客人用了美团券，选**团购套餐**，并在下方选择对应的团购产品。
3. **连坐同开**：如果来了3个朋友一起做手工，把数量改为 `3`，系统自动创建带关联ID的3个计费卡片。

### 4.2 制作过程中
* **核销按钮**：您无需记事本。可在卡片上随时点击 ✘未核销 将其切换为 ✓已核销，帮您记录团购券是否已扫码验券。
* **追加团购**：客人玩一半想再买个券叠加？点击卡片上的【加时 / 加购】，无限往卡片里叠加其他套餐。系统具有**智能抵扣时长算法**。
* **挂账与暂停**：
   - 挂账：客人钱没带够或者去别处，点击结账弹窗里的【挂账等候】。卡片变紫色永久冻结，只等结账。
   - 暂停：客人去吃个饭还要回来？点击【暂离】图标，时间停止转动，客人回来后再点击恢复。

### 4.3 完美结单
1. 点击橙色的 **收款结账**。
2. 系统会弹出一张极为精细的账单明细：列出他的定额时间、暂停了多久、究竟玩了多久、超时计算费率是多少、有没有使用叠加团购优惠。
3. 最下方会显示系统计算出的“一口价”。如遇特殊情况，您可以直接**手动修改实收金额**。
4. 确认后点击【确认收款并结出】。
   > **连坐结账特权**：如果是同行的朋友，弹窗底部会出现蓝色的【一键全结】大按钮，一按即可结清所有同伴！

## 第五章：财务对账

辛苦了一天，点击顶部的【📊 报表与历史】。
- 设定今天的日期范围。
- 顶部会自动计算今日的：“订单总数”、“总接待客流”、“营业实收(拿在手里的现金)”和“已核销团购价值”。
- 老板如果对某一单有异议？找到它！右侧账本清晰记录了每一笔暂离操作和系统推算的原始账面金额，童叟无欺。
- 点击【📥 导出CSV】即可用 Excel 或 WPS 打开查阅。
